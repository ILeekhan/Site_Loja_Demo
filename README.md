# 🛒 Site Loja Demo

Projeto de demonstração de uma loja virtual simples, desenvolvido com tecnologias web básicas, com foco em organização, responsividade e funcionalidade.

---

## 🚀 Funcionalidades
✅ Sistema de cadastro e login de usuários  
✅ Navegação entre páginas (Início, Produtos, Sobre, Contato)  
✅ Catálogo de produtos carregado via arquivo JSON  
✅ Sistema de carrinho de compras (armazenado no navegador)  
✅ Design responsivo para celular, tablet e computador  
✅ Fundo moderno feito com gradiente CSS (sem arquivos de imagem)

---

## 🛠️ Tecnologias Utilizadas
- **HTML5** → Estrutura e semântica das páginas
- **CSS3** → Estilos, gradientes, layout e responsividade
- **JavaScript** → Interatividade, validações e lógica do sistema
- **JSON** → Armazenamento dos dados dos produtos
- **LocalStorage** → Salvar contas de usuário e itens do carrinho
- **Git / GitHub** → Controle de versão e hospedagem do código

---

## 💻 Como executar o projeto
1. Clone ou baixe os arquivos para o seu computador
2. Abra a pasta `Site_Loja_Demo`
3. Dê um duplo clique no arquivo `index.html`
4. O site abrirá diretamente no seu navegador
